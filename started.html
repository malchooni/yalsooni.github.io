지원 증권사 | 환경 구축 | Buycycle 실행 | 구현 예제 | API


  • 지원 증권사
    		•
    • 이베스트투자증권
  • 환경 구축
    		•
    • OS : Window 10
    • JDK 1.8 32bit 설치
    • eBest xing api 설치
    • eBest DevCenter 실행 하여 모든 RES 파일 다운로드
  • Buycycle 실행
    		•
    • GitHub release 페이지에서 buycycle.zip 파일을 다운 받습니다.
    • 압축을 풀고 buycycle/config 디렉토리의 application.yaml 설정 파일을 엽니다.

    • 증권사의 id, 패스워드, 공동인증서 패스워드를 입력 합니다.
    • buycycle/startBuycycle.bat 파일을 실행합니다.

    • http://localhost:7771 브라우저로 접근하여 API 명세 및 테스트가 가능 합니다.
  • 구현 예제
    		•
    • 자바 - HttpURLConnection 활용한 POST 요청
    • 자바 - netty http client POST 요청
  • API
    		•
    Protocol 메시지 URL 내용
    HTTP - GET Json /ebest/description/realtime XAReal TR 목록을 반환 한다.
    HTTP - GET Json /ebest/description/realtime/{trName} XAReal 특정 TR의 명세를 반환 한다.
    HTTP - GET Json /ebest/description/queries XAQuery TR 목록을 반환 한다.
    HTTP - GET Json /ebest/description/queries/{trName} XAQuery 특정 TR의 명세를 반환 한다.
    HTTP - GET Json /ebest/request-messages/{trName} 특정 TR의 요청 샘플 메시지를 반환 한다.
    HTTP - POST
    Content-Type: application/json    		 Json /ebest/query XAQuery (Reqest/Response) TR을 호출 한다.
    WebSocket Json /ebest/real XAReal TR을 호출 한다.

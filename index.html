  • 이게 뭔가요 ?
    		•
    'Buycycle'은 증권사 API를 HTTP Json으로 요청 및 응답 받을 수 있습니다.
    요청 받은 Json 메시지를 증권사 API 양식에 맞게 변환해 주는 자바 기반의 오픈 소스 입니다.
    HTTP RESTful을 제공함으로써 사용자는 개발 언어에 국한되지 않습니다.

    빠른 접근을 위해 'Buycycle' 모듈을 기동 시 웹 화면을 제공하며, 증권사 API에서 제공하는 요청 목록을 열람할 수 있습니다.
    또한 웹 화면에서 테스트도 가능합니다.
    현재는 이베스트투자증권 API만 구현 되어 있으나 기타 증권사도 지원할 예정 입니다.

    본 모듈은 오픈소스로서 잠재적 버그가 존재 할 수 있으며 사용 책임은 사용자에게 있습니다.
  • 아키텍처
    		•
    • HTTP 및 웹소켓 사용을 위해 접근성이 뛰어난 Spring Boot를 활용 합니다.
    • 증권사에서 제공하는 COM버전의 API 사용을 위해 Com4J 라이브러리를 활용 합니다.
    • 실시간성 데이터는 웹소켓을 활용 하여 백엔드와 인터페이스 합니다.
    • 단발성 요청은 HTTP POST method를 사용합니다.
    • 모든 요청 및 응답 메시지는 Json을 활용 합니다.
